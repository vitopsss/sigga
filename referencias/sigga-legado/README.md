# Referencias SIGGA legado

Esta pasta guarda arquivos grandes usados como referencia local, como planilhas antigas, anexos e manuais.

Esses arquivos foram removidos da raiz para deixar o projeto mais limpo. Por serem binarios e pesados, o conteudo desta pasta fica ignorado no Git, exceto este README.

Use esta pasta para:

- planilhas antigas do SIGGA;
- PDFs de anexo/manual;
- bases usadas apenas para consulta, conferencia ou importacao manual.
