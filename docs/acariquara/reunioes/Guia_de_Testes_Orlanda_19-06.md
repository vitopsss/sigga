# Guia Rápido de Testes - SIGGATER (Atualização Junho/2026)

Olá, Orlanda e equipe! Preparadores este guia rápido para que vocês possam testar as novidades do SIGGATER que acabaram de subir para o sistema. 

Nosso foco foi **simplificar a vida do técnico em campo**, garantindo ao mesmo tempo que a exportação de dados para pesquisa saia perfeita.

## 1. O Novo Formulário Unificado (Cadastro + Diagnóstico)
**Como testar:**
1. Acesse o menu **UFPAs**.
2. Clique em **Nova Família** (ou edite uma existente).
3. **O que mudou:** Antigamente os dados eram fragmentados. Agora, em uma única tela, você preenche a Identificação Inicial (vínculo institucional, projeto, data, LGPD) e logo em seguida já responde o Diagnóstico Básico (comunidade, saneamento, meios de comunicação). A rolagem está contínua e mais rápida.

## 2. A "Mágica" do Patrimônio (23 Campos Travados)
Essa foi a alteração feita com base no Formulário 3 físico de vocês!
**Como testar:**
1. Ainda na tela de Cadastro de UFPA, desça até a seção **Patrimônios**.
2. **O que mudou:** Você verá que o sistema agora **traz por padrão 23 itens cravados na tela** (ex: *Bovinos, Aves, Máquinas agrícolas, Pastagens, Extrativismo*, etc).
3. Eles estão bloqueados para alteração de nome (aparecem em cinza). O técnico só precisa bater o olho e preencher a **Quantidade**.
4. *E se a família tiver algo que não está nessa lista de 23 itens?* Basta rolar até o fim da lista e clicar em **"Adicionar Patrimônio"** para incluir um item livremente!

## 3. Módulo Exclusivo de Indicadores (Eixos Social, Ambiental e Econômico)
**Como testar:**
1. Salve a UFPA que você estava editando.
2. Na tela de detalhes da família, clique no botão para **"Preencher Indicadores"** (ou "Editar Indicadores").
3. **O que mudou:** Tiramos o peso da tela inicial e criamos um módulo dedicado só para a tabulação dos Indicadores Completos.

## 4. Formulários Inteligentes (Menos Poluição Visual)
**Como testar:**
1. Dentro do módulo de Indicadores, vá para o **Eixo Econômico**.
2. Marque a opção **"Acessou alguma linha do PRONAF?"** como **SIM**.
3. **O que mudou:** A lista com as linhas específicas do PRONAF (A, B, Mulher, Jovem, etc) só aparece na tela quando você marca SIM. Isso deixa a tela mais limpa para famílias que não acessam a política.
4. Experimente também preencher os campos **"Outros"** nos eixos de Limitações e Ações Potenciais. O texto livre digitado agora é puxado automaticamente na exportação do Excel!

---
**Dica de Ouro:** Não tenham medo de quebrar! Sugerimos criar uma UFPA com o nome "Família Teste" e brincar à vontade com os dados para sentirem a fluidez da nova tela. Qualquer dúvida, pontuem para levarmos para nossa reunião!
